# PGDesk

Modern PG Management Software with Hindi localization, WhatsApp automation, and PWA support.

## Features
- **Hindi/English Localization**: Built-in support for multiple languages including number localization.
- **WhatsApp Automation**: Send rent reminders and updates via WhatsApp.
- **PWA Support**: Installable on Android and iOS devices for a native app experience.
- **Room & Resident Management**: Track occupancy, sections, and beds with ease.
- **Food Management**: Daily food count tracking and reporting.
- **Rent Tracking**: Comprehensive rent history and status (Paid, Pending, Overdue).

## Tech Stack
- **Frontend**: React, Vite, TailwindCSS, Lucide React
- **Backend**: Node.js, Express, MongoDB
- **Automation**: WhatsApp-web.js
